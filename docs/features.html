PHP EMS Tools Package Features

Docs

This is a simple overview of the major features of the PHP EMS Tools package.

  • Password-authenticated administration of roster and user-selectable schedule features.
  • Roster of all members including different member classes
  • Schedule on a month-by-month basis with (currently) two shifts per day
  • Sign-ons on a per shift/day basis via java pop-ups, selectable need for authentication
  • Java pop-ups to edit or remove sign-ons, selectable need for authentication
  • Selectable option to allow only administrators/officers to change schedule entries from previous shifts.
  • Monthly or yearly count of hours by member, highlighting members with less than the required number of hours
  • All options such as member types, required hours, authentication requirements, etc. user-definable in a seperate file.
  • (not yet implemented) - Dispatch Schedule showing each shift in a given week, whether it has a complete crew, and if not, what member types are needed to complete it.
  • Optional email alerts to send the members of a given crew an email or text message when someone signs on, edits their times, or signs off from the particular shift.
  • Easy addition of your site-specific links to the main page.
