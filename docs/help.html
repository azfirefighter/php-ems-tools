PHP EMS Tools Package - Getting Help

Docs

There are a number of ways to get help with PHP EMS Tools:

  1. Check the "help" link which is near the top of each page.
  2. Read through the documentation (what you're looking at right now).
  3. Make sure that your configuration file (custom.php) is setup right. Follow the comments written in the file.
  4. Look for other related problems on the homepage and in the news, forums, and mailing list archives on the SourceForge page.
  5. Subscribe to the php-ems-tools-help mailing list here.
  6. Please refrain from e-mailing me (the developer) with support requests. I only handle requests that are throug the SourceForge page, either with the Support Request feature or through the help mailing list. PLEASE use the help mailing list, as the archives are available to anyone.

Mailing List Guidelines:

  1. Please include the content of version.txt (after the dashed line) with your support request.
  2. Please have read the documentation before posting. If you have a support request after reading the documentation, please include a COMPLETE description of the problem, when it occurs, the exact URL of the page it occurred for and, if appropriate, the entire contents of your custom.php file.
  3. If there is an error involving the database, please include a list of tables by issuing the following two commands in mysql and copying the output: USE php-ems-tools; SHOW TABLES;
