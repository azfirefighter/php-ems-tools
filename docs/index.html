PHP EMS Tools Documentation - Version 3.0

This is the documentation for the second release of php-ems-tools. You can find the newest documentation at Sourceforge or the PHP EMS Tools Homepage.

For updates on bug fixes and new releases, join the sourceforge php-ems-tools-list mailing list HERE. You should probably also consider joining the help mailing list, php-ems-tools-help at sourceforge, which you can do HERE. The help list is a place to ask for help or carry on general discussions about PHP EMS Tools.

For new users, please fill out the online survey so that our development efforts will include features for your organization. This survey asks about basic information such as member types, hours of your shifts, etc.
